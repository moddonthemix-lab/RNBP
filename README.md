A professional R&B production assistant with real instrument sounds, MIDI export, and artist-specific chord progressions.

## Features

### üéµ Real Instrument Sounds
- **Electric Piano** - Rhodes-style FM synthesis with tremolo
- **Warm Pad** - Lush sustained chords
- **Guitar** - Clean pluck synthesis with delay
- **Bass** - Deep sub bass with filter envelope
- **Melody Lead** - Smooth sine with vibrato
- **Drums** - Kick, snare, hi-hats with multiple patterns

### üéº Artist-Specific Progressions
- **PartyNextDoor** - Dark Toronto vibes, minor 9ths, bVI/bVII
- **Bryson Tiller** - Trap soul blend
- **SZA** - Jazz-influenced CTRL era
- **R. Kelly Era** - Classic 90s slow jams
- **Old School** - Jodeci, Boyz II Men, Aaliyah, Usher, TLC
- **Neo Soul** - D'Angelo, Erykah Badu, Maxwell
- **Modern R&B** - Frank Ocean, The Weeknd, Daniel Caesar

### üéπ Music Theory
- Extended chords (7ths, 9ths, 11ths, 13ths)
- Multiple scales (Dorian, Pentatonic, Blues, etc.)
- Intelligent voice leading with inversions
- Swing control for authentic groove

### üìÅ MIDI Export
- Export chords, melody, bass, drums separately
- Export full arrangement
- Compatible with FL Studio, Ableton, Logic, and all DAWs

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- **React** - UI framework
- **Tone.js** - Web Audio synthesis
- **Vite** - Build tool

## Usage

1. Click "Enable Audio" to initialize the sound engine
2. Select a key and BPM
3. Choose an artist style and progression
4. Toggle instruments on/off in the mixer
5. Hit Play to preview
6. Export MIDI files for your DAW
